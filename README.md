# 📚 Sistema de Biblioteca

Sistema desenvolvido com Spring Boot, MongoDB e Thymeleaf, com operações CRUD completas.

## 🚀 Tecnologias
- Java
- Spring Boot
- MongoDB
- Thymeleaf

## 📦 Funcionalidades
- Cadastrar livro
- Listar livros
- Buscar por ID
- Atualizar livro
- Deletar livro

## ▶️ Como executar
1. Rodar MongoDB
2. Executar projeto Spring Boot
3. Acessar: http://localhost:8080
